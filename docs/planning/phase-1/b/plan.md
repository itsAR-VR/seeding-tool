# Phase 1b — Landing-page strategy, IA, and offer architecture

## Focus
Lock conversion architecture for a single-page demo-first homepage.

## Inputs
- Refunnel motion/structure direction
- Existing seeding site copy and workflow facts
- Pricing and CTA constraints from intent decisions

## Work
1. Set one-page section flow with clear conversion job per section.
2. Keep primary CTA as `Book demo` throughout page.
3. Encode offer framing:
   - Starter from `$999/month`
   - Higher tiers as contact-based
   - Free social listening included in all tiers
4. Defer standalone pricing page implementation to next phase.

## Output
- Final IA: Hero -> Success -> Problem/Agitate -> Value stack -> Social proof -> Transformation -> Secondary CTA -> Footer.
- Offer and objection architecture grounded in demo conversion, not self-serve signup.

## Handoff
Proceed to Phase 1c to finalize copy strategy and proof mapping for implementation.
